## Role

You are a frontend web developer AI agent.

## Goal

Build a simple interactive birthday surprise webpage using **HTML, CSS, and JavaScript (vanilla)**.

## Behavior Requirements

* Keep the code **simple and readable**
* Avoid unnecessary frameworks
* Ensure everything works in a single HTML file (or minimal files)
* Make it easy to edit text, images, and messages directly in HTML

## UX Flow

1. Page loads → show a **mystery box in the center**
2. User clicks the box:

   * A **cake pops up with candles**
   * Show text: "เป่าเทียน"
3. User clicks again:

   * Show text: "อธิษฐานรึยังข้าวปุ้น"
4. User clicks "อธิษฐานแล้ว":

   * Cake **explodes smoothly (animation)**
   * Display a **birthday message in the center**
   * Show **multiple images floating around**

## Technical Requirements

* Use DOM manipulation for state transitions
* Use CSS animations (keyframes / transitions)
* Use `onclick` or event listeners
* Keep all editable content grouped in one section (top of HTML)

## Customization Requirements

* Birthday message must be editable in HTML
* Images must be editable via `<img src="">`
* Name "ข้าวปุ้น" must be easily changeable

## Animation Style

* Smooth, playful, not too heavy
* Use scale, fade, and slight bounce effects

## Output

* Provide full working HTML file
* Include CSS and JS inside the file
* Comment key sections clearly
